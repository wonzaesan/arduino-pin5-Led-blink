# Arduino Pin5 Led Blink
## 简介
使用arduino控制引脚5来控制led灯进行时隔1秒的闪烁。
这个是我学习arduino的第一个练习。
## 硬件清单
|元件|规格|数量|
|----|----|----| 
|开发板|arduino uno改良版|1
|杜邦线|  |2
|led||1
色环电阻||1
## 接线方式
pin
5 ->导线->电阻->led阳极（长脚）->led阴极（短脚）->导线->GND
## 使用方法
- 按照接线方式连接好硬件。
- 选择好自己开发板的端口。
- 打开arduino IDE 将代码上传。
